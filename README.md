# Portal Web: PetShop

## Objetivo del Proyecto
El objetivo de este repositorio es documentar y alojar el código fuente del portal web de PetShop. La plataforma busca facilitar la venta de productos para mascotas y la reserva de citas para servicios de juguetes, alimentos entre otros de forma online.

## Contexto del Proyecto

### Tipo de Empresa
El portal ha sido desarrollado para **PetShop**, una tienda especializada en el **cuidado integral de mascotas** que ofrece productos de alimentación, accesorios y servicios de salud animal.

### Funcionalidad y Alcance
La web permite la navegación por categorías de productos, gestión de un carrito de compras y un sistema de registro para el historial clínico de las mascotas. El alcance incluye el catálogo completo de la tienda y un panel de gestión para los empleados de la clínica.

### Usuarios Potenciales
* Dueños de mascotas que buscan productos específicos.
* Veterinarios que necesitan consultar citas programadas.
* Administradores de la tienda para la gestión de stock y pedidos.

## Menú de Navegación Principal
La estructura de navegación de la web es la siguiente:
* Inicio
* Tienda Online
* Servicios Veterinarios
* Peluqueria Canina
* Contacto
